Experiências Profissionais

Analista de Banco de Dados Júnior

Durante minha carreira profissional, tive a oportunidade de trabalhar como Analista de Banco de Dados em uma empresa que prestava serviços à Secretaria da Fazenda do Estado do Pará. Nesse cargo, eu era responsável por garantir que o banco de dados estivesse sempre operando de forma eficiente e segura, para que as equipes de negócio pudessem acessar as informações necessárias para a tomada de decisões, bem como as equipes de desenvolvimento pudessem trabalhar em suas aplicações.

Para isso, eu mantinha contato direto com as equipes de negócio, desenvolvedores, product owners, scrums, entre outros setores importantes. Essa comunicação era essencial para a resolução de problemas que envolviam o banco de dados, já que muitas vezes as demandas dos usuários não eram claras o suficiente e precisávamos trabalhar juntos para entender a melhor forma de atender às necessidades.

Como Analista de Banco de Dados, uma das minhas principais atividades era a realização de análises com SQL. Isso envolvia escrever consultas complexas para extrair informações do banco de dados, cruzando dados de diferentes tabelas e filtrando por critérios específicos. Essa habilidade era fundamental para garantir que as informações estivessem disponíveis para os usuários e que fossem confiáveis e precisas.

Além disso, eu também tinha que manter o banco de dados atualizado e fazer backup regularmente para garantir que não houvesse perda de informações em caso de falhas ou problemas técnicos. Isso exigia atenção constante e dedicação para garantir que os dados estivessem sempre disponíveis e seguros.

Em resumo, minha experiência como Analista de banco de dados me permitiu desenvolver habilidades técnicas importantes em SQL e lidar com a complexidade do gerenciamento de grandes volumes de informações. Além disso, também me permitiu aprimorar minhas habilidades de comunicação e trabalho em equipe para garantir que as necessidades dos usuários fossem atendidas de forma eficiente e eficaz.

Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio foram utilizados, juntamente às ferramentas de desenvolvimento web, como o Streamlit e Github, para desenvolver um Painel Gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.

O resultado final do projeto consiste em um painel hospedado em ambiente Cloud e disponível através de um link web. O painel pode ser acessado por qualquer dispositivo conectado à internet.
